# Gaia Docs

This directory contains Gaia product, architecture, roadmap, findings, and
agent workflow documents.

## Start Here

- `current-capabilities.md`: revision-specific current capability ledger and
  preservation boundary.
- `self-hosting-baseline.md`: fixed GAIA-143 dependency, job, authority, and
  evidence epoch for later comparison.
- `vision.md`: holistic product direction.
- `roadmap.md`: delivery ledger for thin, reversible slices.
- `post-harness-roadmap.md`: implementation track after the real harness loop.
- `operator-model.md`: command authority and read-model decisions.
- `findings.md`: completed slice outcomes and verification evidence.
- `agents/README.md`: Linear-native agent workflow docs for PRDs, issues,
  orchestration, workers, reviewers, and CI/watch loops.

Keep durable product intent in docs or Linear Project/PRD documents, and keep
live Linear workspace state in Linear.
